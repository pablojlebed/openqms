# Calibration Report and Certificate Requirements Procedure

**Procedure number:** 17  
**ISO/IEC 17025 reference:** Clause 7.8

## Purpose

To define the specific requirements for calibration certificates, including mandatory content, statement of conformity, and measurement uncertainty reporting.
